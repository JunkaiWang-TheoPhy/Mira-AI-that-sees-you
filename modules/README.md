# Modules

## Purpose

This directory is the release-side home for Mira's first-party and optional capability modules.

## Owns

- module entrypoints
- module boundary documentation
- release-safe module packaging surfaces

## Does Not Own

- Mira core identity
- long-running service ownership
- raw hardware-side implementation details

## Planned Contents

- flagship module entrypoints
- future wearable, Apple, printer, and messaging module shells
- module-specific integration notes

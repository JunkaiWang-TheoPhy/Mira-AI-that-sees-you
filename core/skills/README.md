# Core Skills

## Purpose

This directory will hold skills that belong to Mira core rather than to optional modules.

## Owns

- core-facing skills
- release-safe skill boundaries

## Does Not Own

- Home Assistant skills
- hardware-specific tools
- module-only behaviors

## Planned Contents

- skill contracts that remain valid without Home Assistant
- release examples for minimal Mira usage

## Current Status

No core skills have been migrated into the release tree yet.
